What is OpenClaw?

🦀

An open-source framework to run a persistent AI agent
on your own machine.


  • Runs 24/7 on a Linux box
  • Connects to your messaging apps
  • Has access to your tools, files & APIs
  • Remembers context across sessions

github.com/openclaw/openclaw • docs.openclaw.ai

Think of it as...

🧠 ChatGPT — but running on your machine

📱 Connected to Telegram, WhatsApp, Discord

🔧 With access to your filesystem, APIs, databases

⏰ That can wake up on its own and do things

💀 SOUL.md — Giving It Personality

# SOUL.md - Who You Are

Be genuinely helpful, not performatively helpful.
Skip the "Great question!" — just help.

Have opinions. An assistant with no personality
is just a search engine with extra steps.

Be resourceful before asking. Try to figure it
out. Read the file. Search for it. THEN ask.

Remember you're a guest. You have access to
someone's life. Treat it with respect.

The AI reads this every session. It shapes behavior.

💡 Lessons Learned

What Works Well

  • File-based memory is surprisingly effective
  • Proactive scheduling (heartbeats + cron) is a game-changer
  • Sub-agent spawning for heavy tasks
  • Free models (Kimi, GLM) handle 90% of tasks
  • Multi-channel: message from anywhere

What's Tricky

  • Context window limits — compaction loses nuance
  • No persistent memory without explicit file writes
  • Image handling quirks (Telegram optimization bugs)
  • WhatsApp gateway disconnections
  • Time tracking needs human confirmation (hours)

Key Insight

The SOUL.md pattern matters more
than the model you use.

Personality, boundaries, and behavior rules
turn a generic LLM into your assistant.
